// Performance: cache expensive queries
function alex_get_featured( $count = 6 ) {
  $key = "featured_posts_{$count}";
  $posts = wp_cache_get( $key );
  if ( ! $posts ) {
    $posts = get_posts( [ 'numberposts' => $count ] );
    wp_cache_set( $key, $posts, '', 3600 );
  }
  return $posts;
}

How I Think

01

I start by asking the uncomfortable questions. What's the actual business goal? Who are the users? What does success look like in 3 months?

Before writing a line of code, I plan the data model, plugin structure, and performance strategy. Architectural decisions made early save weeks of refactoring later.
